# hConfig
This program configures my development environment.



### Usage

- Help
> $ python3 hConfig -h
> $ ./hConfig.py -h

- Installation and COnfiguration
> $ python3 hConfig -a

- Installation
> $ python3 hConfig -i <tools>

- Configuration
> $ python3 hConfig -c <tools>
