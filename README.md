# hSettings
This program can help HoYa's development environment settings.

### Usage
- MAC or LINUX
```
./install.sh all
./setting.sh all
```
- WINDOWS
```
./install.ps1 all
./setting.ps1 all
```

### Eclipse
- Preference File
```
Eclipse_HoYa_Preferences.epf
```
