# hConfig (v2.0)

This program configures my development environment.

### Usage

- Help
> $ python3 hConfig -h
> $ ./hConfig.py -h

- Installation and COnfiguration
> $ python3 hConfig -a

- Installation
> $ python3 hConfig -i <tools>

- Configuration
> $ python3 hConfig -c <tools>

### Support List

##### MAC (Darwin Kernel 17.4.0)

- HomeBrew
- git
- iTerm2
- tmux (tmuxinator)
- zsh (oh-my-zsh)
- Xcode
- gcc-arm-embedded
- st-link

##### Linux (Mint 4.10.0-38-generic)

- git
- clang
- tmux (tmuxinator)
- zsh (oh-my-zsh)
- gcc-arm-embedded
- st-link
