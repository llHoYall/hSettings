# hSettings
This program can help HoYa's development environment settings.

&nbsp;

### Usage

- MAC or LINUX

```sh
./install.sh all
./setting.sh all
```

- WINDOWS

```powershell
./install.ps1 all
./setting.ps1 all
```

&nbsp;

### Eclipse

- Preference File

```sh
Eclipse_HoYa_Preferences.epf
```

&nbsp;
