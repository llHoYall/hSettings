# hConfig (v2.0)

This program configures my development environment.

## Usage

- Help
> $ python3 hConfig -h
> $ ./hConfig.py -h

- Installation and Configuration
> $ python3 hConfig -a

- Installation
> $ python3 hConfig -i <tools>

- Configuration
> $ python3 hConfig -c <tools>

### Available List

* essential (i)
* git (i, c)
* shell (i, c)
* terminal (i, c)
* devtool (i, c)
